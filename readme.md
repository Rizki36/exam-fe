# Booku App
Simple app for frontend test

## How to run
- Install dependencies `yarn install`
- Run in development mode `yarn dev`

## Tech Stack
- Next js
- Typescript
- Redux Toolkit
- SWR
- Framer Motion
- Tailwind Css
